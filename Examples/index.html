groupedOptions custom Knockoutjs binding

Here are three examples of how to implement the groupedOptions custom Knockoutjs binding.

All the examples use the same JavaScript object, demoGroups, which can be easily read by viewing this page's source.

As well as containing the custom binding script, the accompanying JavaScript file, knockout-groupedOptions.js, contains an explanation of each of the parameters and their conventional names.

Conventional implementation

This <select> element is bound to a view-model whose property names follow the conventions defined in groupedOptions.

data-bind attribute

data-bind="groupedOptions: { groups: { coll: Groups }, optionsCaption: '- Please select -', value: SelectedOption }"

JavaScript example file

demo.js

Non-conventional implementation

This <select> element is bound to a view-model whose property names do not follow the conventions defined in groupedOptions.

data-bind attribute

data-bind="groupedOptions: { groups: { coll: Groups, label: 'DisplayText', options: { coll: 'Items', text: 'Display', value: 'Val' } }, optionsCaption: '- Please select -', value: SelectedOption }"

JavaScript example file

demo-non-conventional.js

Non-observable implementation

This <select> element is bound to a view-model whose properties are not observable.

data-bind attribute

data-bind="groupedOptions: { groups: { coll: Groups }, optionsCaption: '- please select -', value: SelectedOption }"
